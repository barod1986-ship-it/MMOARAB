== Description ==

إضافة ووردبريس متخصصة لإدارة وعرض ألعاب MMO بالعربية. تتميز الإضافة بدعم كامل للغة العربية واتجاه RTL، مع تكامل مثالي مع قالب Blocksy.

**الميزات الرئيسية:**

* **إدارة شاملة للألعاب** - نوع محتوى مخصص مع حقول متقدمة
* **4 تصنيفات أساسية** - النوع، الحالة، أسلوب اللعب، المنصات
* **نظام تقييم متقدم** - تقييمات تفصيلية للقصة والجيمبلاي والرسومات والصوت
* **فلترة Ajax متقدمة** - بحث وفلترة فورية مع تحديث URL
* **ربط الأخبار بالألعاب** - ربط المقالات بالألعاب لعرض آخر الأخبار
* **REST API شامل** - واجهة برمجية للاستعلامات الخارجية
* **شورتكود وودجات** - عرض الألعاب في أي مكان بالموقع
* **دعم RTL كامل** - تصميم مثالي للغة العربية
* **تكامل مع Blocksy** - وراثة كاملة لأنماط القالب
* **أداء محسّن** - نظام كاش ذكي وتحميل كسول للصور

**التصنيفات المدعومة:**

* **أنواع الألعاب**: MMORPG, MMOFPS, MOBA, Survival, Sandbox, Action-RPG
* **حالة اللعبة**: قريباً, ألفا, بيتا, وصول مبكر, مُصدر
* **أسلوب اللعب**: PvE, PvP, Co-op, عالم مفتوح, صناعة, غارات
* **المنصات**: PC, PlayStation, Xbox, Nintendo Switch, Mobile

**الحقول المخصصة:**

* معلومات أساسية (المطور، الناشر، تاريخ الإطلاق، الموقع الرسمي)
* محرك اللعبة (Unreal Engine, Unity, CryEngine, Frostbite, مخصص)
* الميزات الرئيسية (حتى 4 ميزات)
* الوسائط (مقاطع يوتيوب، معرض صور حتى 4 صور)
* نظام تقييم شامل (1-10) مع ملاحظات

== Installation ==

1. ارفع مجلد الإضافة إلى `/wp-content/plugins/`
2. فعّل الإضافة من لوحة التحكم
3. اذهب إلى "MOMARAB CORE → إعدادات" للتكوين
4. أضف المصطلحات الأساسية من نفس الصفحة
5. ابدأ بإضافة الألعاب من "الألعاب → إضافة جديدة"

== Frequently Asked Questions ==

= هل تدعم الإضافة RTL والعربية؟ =

نعم، دعم كامل للغة العربية واتجاه RTL مع ملفات ترجمة مدمجة.

= هل تتوافق مع قوالب أخرى غير Blocksy؟ =

الإضافة مصممة خصيصاً لـ Blocksy لأفضل تكامل، لكن يمكن استخدامها مع قوالب أخرى مع تخصيص إضافي.

= كيف أستخدم الشورتكود؟ =

استخدم `[momarab_games limit="6" order="toprated"]` لعرض الألعاب أو `[momarab_game_filter]` للفلترة.

= هل يمكنني تخصيص القوالب؟ =

نعم، يمكنك نسخ ملفات القوالب من `momarab-core/templates/` إلى قالبك الإبن وتخصيصها.

= ما هو الحد الأقصى لعدد الألعاب في الصفحة؟ =

الحد الأقصى 50 لعبة في الصفحة لضمان الأداء الأمثل.

= هل تدعم الإضافة REST API؟ =

نعم، تتضمن واجهة برمجية شاملة على `/wp-json/momarab/v1/games` و `/wp-json/momarab/v1/taxonomies`

= كيف أربط المقالات بالألعاب؟ =

في تحرير المقال، ستجد حقل "ربط باللعبة" في الشريط الجانبي لاختيار اللعبة المرتبطة.
